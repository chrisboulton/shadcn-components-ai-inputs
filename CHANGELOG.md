# Changelog

## 0.1.0 (2025-02-06)

Initial release.

- `AITextbox` — single-line input with AI generation button and optional dropdown for multiple AI actions
- `AITextarea` — multi-line textarea with AI generation button
- `AICombobox` — searchable combobox with AI-powered selection
- Animated gradient border, loading spinner, and one-click undo across all components
